# POS-HJA Frontend

Static React design prototype for the POS-HJA point-of-sale app.

## Run Locally

```powershell
npm install
npm run dev
```

Run as a desktop app during development:

```powershell
npm run app:dev
```

Preview the built desktop app shell:

```powershell
npm run app:preview
```

## Checks

```powershell
npm run lint
npm run build
```
