# Repository Instructions

This repository uses the local `.impeccable` workflow. Before making changes:

1. Read `.impeccable/charter.md` for the project operating principles.
2. Check `.impeccable/tasks.md` for current intent, open work, and decisions.
3. Use `.impeccable/quality-gates.md` to choose verification appropriate to the change.

Keep edits focused, preserve user work, and update the task log when a meaningful decision or verification result should be remembered.
